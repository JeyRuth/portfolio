Analyse de données : Les facteurs influant la performance au marathon

Contextualisation du projet

Ce projet consiste en la création d'une application Shiny permettant d’analyser les 1000 plus gros streamers de jeux vidéo sur Twitch en 2020. L’application est conçue pour aider les investisseurs à prendre des décisions en identifiant les streamers Twitch qui correspondent à leurs critères d’investissement.

Compétences

Il s’agit d’un projet multi-compétence mettant en jeu 3 compétences (dont la compétence du parcours EMS), aussi bien informatiques que statistiques : - Traiter des données à des fins décisionnelles (compétence 1) - Analyser statistiquement des données (compétence 2) - Modéliser les données dans un cadre statistique (compétence 4)

Traiter des données à des fins décisionnelles (compétence 1)

Pour réaliser ce projet, j’ai développé une application Rshiny. Il s’agit d’une application web interactive développée à l’aide du langage R et du framework Shiny. Ainsi, elle permet de visualiser des résultats de manière interactive.

Tout d’abord, je me suis employé à du brainstorming pour définir l’interface utilisateur (UI). De cette façon, je pouvais réaliser une application innovante respectant le cahier des charges fourni, tout en apportant mes idées personnelles. J’ai réalisé 5 onglets (dont 1 avec 2 sous-onglets) qui sont les suivants : - Twitch - Dashboard - Classement - Analyse multivariée o ACP o K-means - Données

Analyser statistiquement des données (compétence 2)

L'analyse univariée consiste à examiner les caractéristiques d'une seule variable à la fois. Ainsi, j’ai créé un premier diagramme en barres verticales qui représente le taux de visionnage des streamers. Ce type de graphique permet de visualiser rapidement la tendance générale. Nous savons tout de suite si un streamer fait mieux que l’autre. De la même manière, j’ai aussi représenté le taux d’engagement des abonnées. L’analyse bivariée consiste à examiner plusieurs variables en même temps. J’ai réalisé un diagramme en barres compilés représentant la part de streamer pour chaque langue dans chaque catégorie de visionnage.

Modéliser les données dans un cadre statistique (compétence 4)

Pour réaliser ce projet, j’ai réalisé une analyse multivariée. C’est une méthode d'analyse statistique qui permet d'étudier simultanément les relations entre plusieurs variables. Les techniques que j’ai utilisées sont l’analyse en composantes principales et la classification.

J’ai réalisé un heatmap des corrélations, pour visualiser les variables corrélées entre elles. Ensuite, j’ai séparé les données quantitatives du jeu de données et je les ai standardisé afin de réaliser une analyse en composantes principales. Cette méthode permet de réduire la dimensionnalité d'un ensemble de données en identifiant les combinaisons linéaires des variables originales qui expliquent le mieux la variance des données. Ainsi, l’investisseur peut rapidement visualiser les meilleurs streamers dans un domaine précis.
