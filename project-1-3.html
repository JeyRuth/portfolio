Description et prévision de données temporelles

Contextualisation du projet

L'objectif de ce projet est d'analyser une série temporelle de production d'électricité par combustion du charbon entre 2001 et 2021, provenant de l'Agence américaine d'information sur l'énergie (EIA). L'analyse vise à fournir des prévisions pour guider les décisions d'investissement et les stratégies énergétiques à long terme. Il s'agit d'un projet réalisé à trois, je vais vous présenter les parties que j'ai faites.

Compétences

Il s’agit d’un projet multi-compétence mettant en jeu 3 compétences (dont la compétence du parcours EMS), aussi bien informatiques que statistiques : - Traiter des données à des fins décisionnelles (compétence 1) - Analyser statistiquement des données (compétence 2) - Modéliser les données dans un cadre statistique (compétence 4)

Traiter des données à des fins décisionnelles (compétence 1)

J'ai débuté par explorer les caractéristiques temporelles de la série pour identifier les tendances, les variations saisonnières et les fluctuations aléatoires.

J'ai également détecté les points aberrants et les valeurs manquantes dans les données pour préparer la série pour l'analyse.

Analyser statistiquement des données (compétence 2)

J'ai utilisé des techniques statistiques pour modéliser la tendance de la série, telles que les moyennes mobiles et les moyennes annuelles. J'ai également calculé les coefficients saisonniers pour mettre en évidence les variations saisonnières dans la production d'électricité.

Modéliser les données dans un cadre statistique (compétence 4)

J'ai appliqué la méthode de Holt-Winters pour prévoir la production future d'électricité. Cette méthode suppose une relation entre les observations antérieures et les prévisions futures, en attribuant des poids décroissants aux observations antérieures. Enfin, j'ai comparé les résultats des différentes méthodes de prévision pour sélectionner le modèle le plus approprié, en l'occurrence le modèle ARMA qui a été réalisé par un membre du groupe.
